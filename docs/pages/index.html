Key Features

9 Algo Strategies

LIMIT, TWAP, VWAP, POV, ICEBERG, SNIPER, ARRIVAL_PRICE, IS, and MOMENTUM. Each runs as an independent microservice consuming from the Redpanda message bus.

Real + Simulated Data

Built-in market simulator generates GBM price paths, seeded with real prices from Alpha Vantage, Polygon, Tiingo, and FRED data feeds.

Fixed Income & Derivatives

Bond pricing, yield curve construction, duration ladders, spread analysis, volatility surfaces, and Black-Scholes option pricing.

FIX Protocol

Full FIX 4.2 exchange simulation with TCP gateway, session management, execution reports, and a FIX archive for trade reconstruction.

Intelligence Pipeline

Feature engine, signal engine, and recommendation engine work together to surface trading insights. Optional LLM advisory via Ollama.

RBAC & OAuth2

Role-based access control with 4 trader personas: high-touch, algo, fixed income, and read-only analyst. OAuth2 + PKCE authentication.

Multi-Workspace UI

Draggable FlexLayout panels with named workspaces: trading, algo, FI analysis, admin. Linked channels for cross-panel communication.

Dark Pool & CCP

Simulated dark pool crossing network and central counterparty clearing service for realistic post-trade workflows.

Electron Desktop

Cross-platform desktop app with pop-out panels, native menus, and linked windows. Builds for macOS, Windows, and Linux.

Session Replay

Record and play back user sessions with rrweb. DOM mutations, clicks, and scrolls captured with full input masking. Admin-controlled toggle with timeline playback.

Architecture

Event-driven microservices connected by a Redpanda (Kafka-compatible) message bus.
